# Books I have Read

## Description
The purpose of this project is to hone database skills, while building a server. 

## Tools
- EJS
- Node JS, & Express JS
- Postgres SQL
- CSS
- JavaScript

## Basic Features:
- Users can view all books I have read.
- I can add new books.
- I can edit books already added.
- I can also delete a book entry
